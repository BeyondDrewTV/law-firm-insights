# Clarion — Phrase Dictionary Calibration Log
# Version: 1.0 | Human-maintained. No agent may write to this file directly.

---

## Purpose

This file tracks all proposed and approved changes to Clarion's phrase dictionary.
Proposals are generated by the Phrase Dictionary Calibration Analyst.
All changes require human review and approval before implementation.

---

## Log Format

Each entry must include:
- Date proposed
- Proposed by (agent or human)
- Change type: ADD | REMOVE | RECLASSIFY
- Phrase
- Current mapping (if applicable)
- Proposed mapping
- Rationale
- Status: PROPOSED | APPROVED | REJECTED
- Approved by (human name, if approved)
